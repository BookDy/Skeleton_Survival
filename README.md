# 스켈레톤 서바이벌
스켈레톤 서바이벌 문서입니다.
<br />
<br />
# 1. 프로젝트 소개
  ### 프로젝트 개요
  - 뱀파이어 서바이벌에서 영감을 받아 제작된 스켈레톤 서바이벌입니다.
  - 몬스터를 쓰러뜨려 레벨업을 하면서 플레이어가 성장하고 정해진 웨이브 내를 최대한 빨리 깨는 것이 목표인 게임입니다.
  <br />
  
  ### 개발 인원
  - 현정민(1인)
  <br />

  ### 사용 기술
  - 언리얼 엔진(c++과 블루 프린트)
  - 블랜더(모델링 수정)
  <br />
